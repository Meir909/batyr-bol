# Настройка интеграции с Gemini API

## Получение API ключа

1. Перейдите на сайт [Google AI Studio](https://aistudio.google.com/)
2. Войдите в свою учетную запись Google
3. Нажмите на кнопку "Get API key" или "Получить ключ API"
4. Создайте новый ключ API или используйте существующий
5. Скопируйте ваш API ключ

## Настройка API ключа в проекте

### Вариант 1: Использование .env файла (рекомендуется)

1. Создайте файл `.env` в корневой директории проекта (рядом с `server.py`)
2. Добавьте в него строку:
   ```
   GEMINI_API_KEY=ваш_реальный_api_ключ_здесь
   ```

### Вариант 2: Прямая замена в коде

1. Откройте файл `game_integration.js`
2. Найдите строку:
   ```javascript
   this.geminiApiKey = 'YOUR_GEMINI_API_KEY';
   ```
3. Замените `'YOUR_GEMINI_API_KEY'` на ваш реальный API ключ:
   ```javascript
   this.geminiApiKey = 'ваш_реальный_api_ключ_здесь';
   ```

## Проверка работы

После настройки API ключа:
1. Запустите сервер: `python server.py`
2. Откройте браузер и перейдите по адресу `http://localhost:8000`
3. Перейдите в игру и войдите в систему
4. Нажмите кнопку "Получить миссию"
5. Если интеграция работает корректно, вы увидите контент, сгенерированный с помощью Gemini API

## Возможные проблемы

### Ошибка 400 или 403
- Проверьте правильность API ключа
- Убедитесь, что ключ не содержит лишних пробелов

### Ошибка сети
- Проверьте подключение к интернету
- Убедитесь, что брандмауэр не блокирует запросы

### Фолбэк на статический контент
Если API не отвечает, система автоматически использует статический контент из файла.

## Безопасность

⚠️ **Важно**: Никогда не публикуйте ваш API ключ в открытом виде в репозитории или в публичных местах.
Используйте переменные окружения или другие безопасные методы хранения ключей.

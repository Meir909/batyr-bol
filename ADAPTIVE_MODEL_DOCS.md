# Адаптивная модель обучения BATYR BOL

## Обзор

Адаптивная модель обучения предназначена для генерации персонализированного образовательного контента о казахской истории и языке. Модель автоматически адаптирует сложность материалов и вопросов в зависимости от уровня знаний пользователя.

## Основные возможности

### 1. Генерация контента
- Создание небольших историй и текстов о казахской истории и языке
- Адаптация контента под уровень пользователя (начальный, средний, продвинутый)
- Разнообразные категории: история, язык, грамматика

### 2. Создание вопросов
- Генерация вопросов на основе предоставленного контента
- Адаптация сложности вопросов (открытые, с выбором ответа, аналитические)
- Автоматическая генерация вариантов ответов для тестовых заданий

### 3. Адаптация уровня
- Анализ ответов пользователя в реальном времени
- Автоматическая настройка уровня сложности
- Мягкая обратная связь при неправильных ответах

### 4. Обратная связь
- Поддержка пользователей при затруднениях
- Поощрение за правильные ответы
- Индивидуальные подсказки на основе истории ответов

## Структура модели

### Компоненты
1. **База контента** - коллекция статей о казахской истории и языке
2. **Шаблоны вопросов** - набор вопросов различной сложности
3. **Система оценки** - алгоритмы проверки ответов
4. **Адаптер уровня** - механизм адаптации под пользователя

### Уровни сложности
- **Начальный (beginner)** - базовые факты и простые вопросы
- **Средний (intermediate)** - аналитические вопросы и сложные концепции
- **Продвинутый (advanced)** - открытые вопросы и глубокий анализ

## Интеграция с Telegram-ботом

### Основные команды
- `/missions` - получение персонализированных заданий
- `/answer` - отправка ответов на вопросы
- `/profile` - просмотр прогресса и уровня

### Адаптивные функции
- Автоматический подбор контента по уровню пользователя
- Персонализированные вопросы на основе истории ответов
- Динамическая настройка сложности

## Примеры использования

### Генерация контента
```python
# Получение контента для пользователя среднего уровня по истории
content = model.get_adaptive_content("intermediate", "history")
```

### Создание вопросов
```python
# Генерация 3 вопросов на основе контента
questions = model.generate_questions(content, "intermediate", 3)
```

### Оценка ответов
```python
# Оценка ответа пользователя
is_correct, feedback, new_level = model.evaluate_answer(
    question, user_answer, user_history
)
```

## Расширение модели

### Добавление нового контента
1. Добавить запись в `content_database`
2. Указать уровень сложности
3. Определить ключевые факты и слова

### Добавление новых шаблонов вопросов
1. Добавить шаблоны в `question_templates`
2. Указать уровень сложности
3. Убедиться в соответствии типам вопросов

### Настройка обратной связи
1. Добавить сообщения в `feedback_messages`
2. Классифицировать по типам (поощрение, конструктивная, поддерживающая)

## Часто задаваемые вопросы

### Почему модель не повышает мой уровень?
Уровень повышается при стабильных правильных ответах (точность > 80%).

### Как добавить новый контент?
Смотрите раздел "Расширение модели" в этой документации.

### Можно ли использовать модель вне Telegram?
Да, модель является независимым модулем и может интегрироваться в другие приложения.

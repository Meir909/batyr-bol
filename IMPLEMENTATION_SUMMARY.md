## Запуск и использование

### Запуск сервера
```bash
python server.py
```

### Настройка Gemini API
1. Создайте файл `.env` на основе `.env.example`
2. Добавьте ваш API ключ:
   ```
   GEMINI_API_KEY=ваш_реальный_api_ключ
   ```
3. Перезапустите сервер

### Использование
1. Откройте браузер и перейдите по адресу `http://localhost:8000`
2. Нажмите "Начать играть в браузере" или "Играть в Telegram"
3. Для веб-интерфейса:
   - Зарегистрируйтесь или войдите в систему
   - Нажмите "Получить миссию"
   - Прочитайте контент и нажмите "Ознакомился"
   - Ответьте на вопросы
4. Для Telegram бота:
   - Откройте Telegram
   - Найдите бота @batyr_bol_bot
   - Начните диалог с командой /start

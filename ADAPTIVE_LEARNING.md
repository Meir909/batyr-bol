# Адаптивная система обучения BATYR BOL

## Обзор

Адаптивная система обучения BATYR BOL представляет собой интеллектуальную модель, которая генерирует образовательный контент о казахской истории и языке, задает персонализированные вопросы и адаптирует сложность материалов в зависимости от уровня знаний пользователя.

## Основные компоненты

### 1. Модель адаптивного обучения (`learning_model.py`)

#### Основные функции:
- **Генерация контента**: Создание текстов о казахской истории и языке трех уровней сложности
- **Адаптивные вопросы**: Формирование вопросов, соответствующих уровню пользователя
- **Оценка ответов**: Интеллектуальная проверка правильности ответов
- **Адаптация сложности**: Автоматическая настройка уровня сложности
- **Обратная связь**: Мягкие поддерживающие сообщения при ошибках

#### Уровни сложности:
1. **Beginner** (Начальный) - Базовые факты и простые вопросы
2. **Intermediate** (Средний) - Анализ и сравнение исторических событий
3. **Advanced** (Продвинутый) - Критическое мышление и аналитические задания

### 2. Интеграция с Telegram-ботом (`enhanced_bb_bot.py`)

#### Новые возможности:
- **Персонализированные миссии**: Контент, адаптированный под уровень пользователя
- **Интеллектуальная оценка**: Анализ ответов с учетом контекста
- **Прогрессивное обучение**: Автоматическое повышение/понижение сложности
- **История обучения**: Отслеживание прогресса каждого пользователя

## Как работает система

### 1. Инициализация пользователя
```
При первом взаимодействии:
- Уровень сложности: beginner
- История обучения: пустая
- XP: 0
```

### 2. Генерация контента
```
1. Выбор типа контента (история/язык)
2. Подбор материала согласно уровню пользователя
3. Создание персонализированных вопросов
4. Представление контента пользователю
```

### 3. Обработка ответов
```
1. Получение ответа пользователя
2. Интеллектуальная оценка правильности
3. Начисление XP за правильные ответы
4. Адаптация уровня сложности
5. Предоставление поддерживающей обратной связи
```

### 4. Адаптация сложности
```
Повышение уровня:
- Высокая точность ответов (>80%)
- Последовательные правильные ответы
- Готовность к более сложным материалам

Понижение уровня:
- Низкая точность ответов (<50%)
- Повторные ошибки
- Необходимость в более простом материале
```

## Типы вопросов

### Beginner (Начальный уровень)
- Фактические вопросы (когда, кто, где)
- Вариативные тесты (выбор одного правильного ответа)
- Простые аналитические задания

### Intermediate (Средний уровень)
- Причинно-следственные вопросы
- Сравнительный анализ
- Упорядочивание событий
- Множественный выбор

### Advanced (Продвинутый уровень)
- Аналитические эссе
- Критические оценки
- Связывание с современностью
- Аргументированные ответы

## Техническая реализация

### Структура данных пользователя
```python
user = {
    "xp": 0,                      # Очки опыта
    "level": 1,                   # Уровень пользователя
    "difficulty_level": "beginner", # Уровень сложности контента
    "learning_history": [],       # История обучения
    "current_content": {},        # Текущий контент
    "current_questions": []       # Текущие вопросы
}
```

### Структура контента
```python
content = {
    "title": "Название темы",
    "text": "Образовательный текст",
    "difficulty": "beginner|intermediate|advanced",
    "key_facts": ["Факт 1", "Факт 2"],
    "keywords": ["ключевое слово 1", "ключевое слово 2"]
}
```

### Структура вопросов
```python
question = {
    "id": "уникальный_идентификатор",
    "text": "Текст вопроса",
    "type": "open|choice|ordering",
    "difficulty": "beginner|intermediate|advanced",
    "correct_answer": "Правильный ответ",
    "options": ["вариант 1", "вариант 2"]  # Для тестов
}
```

## Преимущества системы

1. **Персонализация**: Каждый пользователь получает контент своего уровня
2. **Адаптивность**: Сложность автоматически подстраивается под способности
3. **Поддержка**: Мягкая обратная связь помогает не бояться ошибок
4. **Прогресс**: Визуальное отслеживание достижений и роста
5. **Многообразие**: Разные типы контента и вопросов

## Возможности для расширения

1. **Расширенная база контента**: Больше историй и языковых материалов
2. **Улучшенная оценка NLP**: Более точный анализ ответов
3. **Мультимедиа**: Интеграция изображений и аудио
4. **Социальные функции**: Совместные задания и соревнования
5. **Статистика**: Подробная аналитика прогресса

## Запуск системы

1. Установите необходимые зависимости:
```bash
pip install python-telegram-bot
```

2. Замените токен бота в файле `enhanced_bb_bot.py`

3. Запустите бота:
```bash
python enhanced_bb_bot.py
```

4. Для тестирования модели:
```bash
python test_learning_model.py
```
